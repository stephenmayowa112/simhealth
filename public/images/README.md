## Image Guidelines

1. **Format**: Use JPG for photos, PNG for graphics with transparency, WebP for optimized web delivery
2. **Optimization**: Compress images before uploading to reduce file size
3. **Alt Text**: Always provide descriptive alt text in the component props
4. **Responsive**: Images will be automatically optimized by Next.js Image component
5. **Naming**: Use lowercase with hyphens (e.g., `health-program.jpg`)
